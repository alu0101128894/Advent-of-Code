# Advent of Code 2021

### Ejecución del programa
Para compilar los archivos utilizamos el 
> ./bin/nombre_ejecutable

### Clasificación Archivos
- `/bin` = Ejecutables
- `/include` = Headers (.h)
- `/src` = (.cc)
- `Makefile` = Archivo para compilar (make)
